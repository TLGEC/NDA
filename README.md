# James Cooling NDA Page

Upload `index.html` to the root of the GitHub Pages repository.

Live page should be:
https://tlgec.github.io/NDA/

This version is deliberately simple:
- No Make webhook
- No backend
- No automatic hidden storage
- No attempted email attachment from the browser

The button creates and downloads `[Full Name] NDA.pdf`, then opens an email draft to:
james.cooling@gmail.com

The signer should attach the downloaded PDF before sending.
